# ROADMAP.md - Pushpaka Product Roadmap

## Vision

Pushpaka aims to be the leading self-hosted, enterprise-grade deployment platform that brings Vercel/Render/Railway-like experience to your infrastructure.

## Current Status: v1.0.0 ✅

**Release Date:** March 17, 2026

### Completed
- ✅ Core deployment pipeline
- ✅ Git integration (public & private repos)
- ✅ Auto Dockerization (15+ frameworks)
- ✅ Real-time log streaming
- ✅ Traefik v3 + Let's Encrypt SSL
- ✅ Multi-project & multi-user support
- ✅ Dashboard with dark/light theme
- ✅ Professional website with documentation
- ✅ Helm charts for Kubernetes
- ✅ GitHub Actions CI/CD
- ✅ Docker & single-binary deployment modes

## Next Milestones

### Q2 2026 - v1.1.0 🎯

**Theme:** Operational Excellence

Features:
- **Scheduled Deployments** - Schedule deployments for specific times
- **Email Notifications** - Deployment status notifications
- **Advanced API Key Management** - API key rotation and usage analytics
- **Mobile Dashboard** - Progressive Web App (PWA) support
- **Environment Cloning** - Clone environment variables between projects
- **Webhooks** - Pre/post-deployment webhooks
- **Deployment Strategies** - Blue-green, canary deployments

### Q3 2026 - v1.2.0 🚀

**Theme:** Enterprise Features

Features:
- **Multi-Region Deployments** - Deploy to multiple regions with traffic steering
- **Database Migration Tools** - Automated backup, restore, migration
- **Advanced RBAC** - Fine-grained permission control
- **Git SSH Authentication** - SSH key-based git access (in addition to PAT)
- **Deployment Templates** - Pre-configured deployment templates
- **Cost Analytics** - Resource usage and cost estimation
- **AI Deployment Optimization** - Auto-optimize build and deployment processes

### Q4 2026 - v1.3.0 🔧

**Theme:** Extensibility

Features:
- **Custom Build Runners** - Bring your own build infrastructure
- **Plugin System** - Pluggable authentication, storage, logging
- **Audit Logging** - Comprehensive audit trail
- **SAML 2.0 Support** - Enterprise SSO integration
- **OpenTelemetry** - Full observability stack
- **Cost Optimization** - Automatic resource optimization
- **Custom Operators** - Kubernetes operator for advanced deployments
